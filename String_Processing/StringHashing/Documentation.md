# String Hashing

## 1. Introduction
String hashing is a method that allows correct string comparison (with high probability) in constant time **O(1)** (assuming we already have the hashes).

It requires a **hash function**: a mathematical function such that for any strings `S` and `T`:

- If `S == T` ⇒ `hash_function(S) == hash_function(T)`
- The probability that `hash_function(S) == hash_function(T)` ⇒ `S == T` is nearly **1**.

---

## 2. Example of a Hash Function

```cpp
int count_unique_substrings(string const& s) {
    int n = s.size();

    const int p = 31;
    const int m = 1e9 + 9;
    vector<long long> p_pow(n);
    p_pow[0] = 1;
    for (int i = 1; i < n; i++)
        p_pow[i] = (p_pow[i-1] * p) % m;

    vector<long long> h(n + 1, 0);
    for (int i = 0; i < n; i++)
        h[i+1] = (h[i] + (s[i] - 'a' + 1) * p_pow[i]) % m;

    int cnt = 0;
    for (int l = 1; l <= n; l++) {
        unordered_set<long long> hs;
        for (int i = 0; i <= n - l; i++) {
            long long cur_h = (h[i + l] + m - h[i]) % m;
            cur_h = (cur_h * p_pow[n-i-1]) % m;
            hs.insert(cur_h);
        }
        cnt += hs.size();
    }
    return cnt;
}
```
